NHS Health widget configuration

The NHS Choices Health A-Z widget can be configured using this utility. In order to build your widget, you'll need to select a topic, and then add sections, such as Diagnosis, Causes and treatments, which will make up the contents of your widget.

To get started, please choose from the two options below. A-Z will produce an alphabetised list of conditions, which you can select from.Subject displays topics according to category - for example, the subject Pregnancy contains topics such as IVF, Labour and Morning Sickness.

Each condition is created in a separate widget. If you want to create multiple topics, you'll need to build an individual widget for each topic.
